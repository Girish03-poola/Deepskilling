Q1. What is recursion?

Recursion is a technique where a method calls itself until a base condition is reached.

Q2. Why is recursion used?

It simplifies problems that can be broken into smaller, similar subproblems.

Q3. What is the base case?

The base case is years == 0, where the method returns the current value without making further recursive calls.

Q4. What is the time complexity?

O(n) because one recursive call is made for each year.

Q5. How can recursion be optimized?

By using an iterative loop, memoization (when applicable), or the direct compound interest formula to avoid excessive recursive calls.
